Fennec

Minimal, customizable Firefox/LibreWolf setup with a Zen Browser sidebar workflow. No fork, no build — just CSS.

Install GitHub

Features

Enhanced Sidebery

Urlbar lives in the sidebar, expands on focus

Zen Mode

Toggle the sidebar to hide the UI and maximize focus

Minimal Chrome

Only the essentials, built for keyboard-driven use

Theme Support

System themes and user-created Firefox themes supported

Quick Install

Install Sidebery, then run:

macOS / Linux curl -fsSL https://raw.githubusercontent.com/tompassarelli/fennec/main/install.sh | bash
Windows (PowerShell) irm https://raw.githubusercontent.com/tompassarelli/fennec/main/install.ps1 | iex

LibreWolf? Add --librewolf to either command. See the full install guide.

Why Fennec?

  • No browser fork — just CSS
  • Update-safe — your tweaks survive Fennec updates
  • Works on Firefox and LibreWolf
